这一天背后的 3 个原则

    想要 AI 团队按你**这一周的目标**给你排你今天的一天？

    这是什么

    不是一个 AI——是一组 Agent

    🥗 饮食 Agent

    只盯一件事——你今天吃什么。它知道你的目标宏量、你常吃的菜、你训练前后该补什么。它不评论你的训练，也不给睡眠建议。

    🏋️ 运动 Agent

    只盯一件事——你今天该练什么。它知道你昨晚 HRV 有没有低、你这周训练量、你的目标周（减脂还是 peak）。

    😴 睡眠 Agent

    只盯一件事——你的恢复。它发现 HRV 异常会主动给运动 Agent 发消息：「今天降强度。」

    🧭 导航员

    3 个专家意见相反时，它决定**谁说了算**——根据你这一周的目标。然后写出一份计划 + **一句取舍**。

    这是 Mana 的 demo——24 小时盯着你的 AI 团队，根据你的画像和目标排你今天的一天。每条建议都告诉你**为什么**，因为减肥、备赛、增肌——每一条决定都需要上下文。
